# MISSPipe

**MISSAVやKISSJAVなどを快適に閲覧するためのオープンソースAndroidアプリ**

PipePipeをベースにした、**R18動画専用フォーク**です。  
広告をブロックし、快適に動画を検索・視聴できます。

## 特徴

- R18専用最適化（非公式API + PipePipeの強力なエンジン）
- 広告ブロック
- バックグラウンド再生対応
- オープンソース（GPL-3.0）

## インストール方法

1. [Releases](https://github.com/Kdroidwin/MISSPipe/releases)から最新のAPKをダウンロード
2. または自分でビルドして使用

**注意**: Google Playには公開されていません。信頼できるソースからのみAPKを入手してください。

## ビルド方法

```bash
# リポジトリをクローン
git clone https://github.com/Kdroidwin/MISSPipe.git
cd MISSPipe

# PipePipe部分の準備（必要に応じて）
cd PipePipe-main/PipePipe-main
# Android Studioで開くか、コマンドでビルド
```

詳細は `PipePipe-main/PipePipe-main` フォルダ内のドキュメントを参照。[主なコード](https://github.com/Kdroidwin/MISSPipe/tree/main/PipePipe-main/PipePipe-main/PipePipeExtractor/extractor/src/main/java/org/schabi/newpipe/extractor/services/missav)

## 使用上の注意

- このアプリは**非公式**です。MISSAVの利用規約に違反しない範囲で自己責任でご使用ください。
- アダルトコンテンツを扱うアプリです。18歳未満の方は使用しないでください。
